SVG-декорация · spin-cw / spin-ccw · скорость через --sd
Правила:
· Сплошное кольцо — без класса, статичное
· Пунктирное — class="spin-cw" или spin-ccw"
· Скорость — style="--sd:90s" (больше = медленнее)
· Позиция — через cx/cy, НЕ через transform="matrix(...)"
· Добавить: transform-box:fill-box; transform-origin:50% 50%
Анимация входа · класс .a · задержка через --d
<h2 class="a" style="--d:0s"> — появляется сразу
<p class="a" style="--d:.2s"> — через 0.2s
<div class="a" style="--d:.35s"> — через 0.35s

Анимация запускается только когда слайд получает класс .active
